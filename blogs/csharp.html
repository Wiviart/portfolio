The Four Pillars of OOP
  • Abstraction - Use interfaces or abstract classes to define behavior.
  • Encapsulation - Hide fields and use properties to access them.
  • Inheritance - Derive new classes from base classes.
  • Polymorphism - Override or implement methods for different behaviors.
Example: Interface vs Abstract Class 
interface IAnimal {
    void MakeSound();
}

class Dog : IAnimal {
    public void MakeSound() {
        Console.WriteLine("Woof!");
    }
}
abstract class Animal {
    public abstract void MakeSound();
    public void Move() => Console.WriteLine("Moves");
}

class Dog : Animal {
    public override void MakeSound() {
        Console.WriteLine("Woof!");
    }
}
